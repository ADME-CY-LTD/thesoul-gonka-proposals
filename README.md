# TheSoul × Gonka governance proposals

Six independent Cosmos SDK governance proposals submitted to the Gonka chain by TheSoul Group — four paid in IBC USDT (via the community USDT vault's `withdraw_ibc`), two in native GNK (via `MsgCommunityPoolSpend` from the community pool). Each pays a single tranche to TheSoul on proposal pass.

Each `proposals/*.json` is the exact payload for `inferenced tx gov submit-proposal`. Full context for every offer — scope, deliverables, and the on-chain message — lives in its discussion thread, linked from the JSON's `metadata` field.

Master overview: [#7](https://github.com/ADME-CY-LTD/thesoul-gonka-proposals/issues/7).
